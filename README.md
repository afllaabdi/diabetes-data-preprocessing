# Pima Indians Diabetes Data Preprocessing

This project focuses on data preprocessing and exploratory analysis using the **Pima Indians Diabetes dataset**.

## Workflow

1. Dataset loading using Pandas
2. Data inspection and preview
3. Handling missing values
4. Median imputation for numerical columns
5. Data visualization using Seaborn
6. Feature scaling using MinMaxScaler
7. Dataset statistics and summary

## Technologies

Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn

## Goal

Understanding the importance of data preprocessing before building machine learning models.
